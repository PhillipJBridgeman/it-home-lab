## Summary
The client approved a memory (RAM) upgrade to improve system performance for CompTIA A+ certification studies and general system responsiveness. The system previously operated with limited memory, impacting multitasking and learning workflows. This ticket documents the upgrade, validation, and outcomes.

---

## Business / Technical Justification
The upgrade was approved to:
- Support **CompTIA A+ Core 1 and Core 2 hands-on learning**
- Improve system responsiveness while studying, browsing documentation, and running operating systems
- Extend the usable lifespan of legacy hardware for educational and lab purposes
- Simulate real-world IT support scenarios involving hardware upgrades and BIOS validation

---

## Pre-Upgrade System Details
- **Motherboard BIOS:** N68C-S UCC P1.40
- **Processor:** AMD Athlon II X2 255 (64-bit)
- **Processor Speed:** 3.10 GHz
- **Memory (Before):**  
  - 4 GB DDR3 (2 × 2 GB)  
  - Dual-channel mode  
- **Operating System:** Ubuntu Linux (Home Lab System)

---

## Change Performed
- Removed existing 2 × 2 GB DDR3 modules
- Installed upgraded memory:
  - **2 × 4 GB DDR3 modules**
- Ensured modules were seated correctly in dual-channel slots
- Powered on system and entered BIOS for validation

---

## Post-Upgrade Validation
- **Total Memory Detected:** 8 GB (8192 MB)
- **Memory Configuration:** Dual-Channel Mode
- **Memory Speed:** DDR3-1333 (667 MHz reported)
- **Shared Memory:** 256 MB
- **BIOS Recognition:** Successful
- **System Boot:** Successful
- **Stability Testing:** No POST errors or instability observed

## Issues Encountered
- None

## Resolution
The memory upgrade was completed successfully. The system now operates with doubled RAM capacity, improving performance and usability for learning, lab work, and daily tasks.
